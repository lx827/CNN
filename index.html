PyTorch CNN MNIST 深度学习

📊 项目概述

这是一个使用卷积神经网络 (CNN) 进行手写数字识别的深度学习项目。项目基于 PyTorch 框架，使用经典的 MNIST 数据集进行训练和评估。包含两个版本的训练脚本，从基础到优化，适合学习和实践深度学习模型开发。

99%+ 测试准确率
60,000 训练样本
10,000 测试样本
2 版本脚本

🏗️ 模型架构

V1 基础版 (MNISTCNN)

输入(1×28×28) → Conv(32) → BN → ReLU → MaxPool → Conv(64) → BN → ReLU → MaxPool → Flatten → Linear(128) → ReLU → Dropout(0.5) → Linear(10)

V2 优化版 (MNISTCNNImproved)

输入(1×28×28) → [Conv(32)→BN→ReLU→Conv(32)→BN→ReLU→MaxPool→Dropout] → [Conv(64)→BN→ReLU→Conv(64)→BN→ReLU→MaxPool→Dropout] → [Conv(128)→BN→ReLU] → Flatten → Linear(256) → ReLU → Dropout(0.3) → Linear(10)
💡 优化亮点：V2 版本采用更深的网络结构（3个卷积块 vs 2个），引入数据增强、学习率调度器、早停机制和权重衰减，有效提升了模型泛化能力。
