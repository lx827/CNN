## 4.8 结果评述

从以上实验结果可以看出：

1. **恒速轴承二分类（HUSTbear 99样本）**：
   - 🏆 **VMD 以 88.89% 最佳**，比第二名 DWT（82.83%）高 6pp
   - 标准包络和 EMD 并列第三（81.82%），MED/MCKD（78.79%）
   - Ensemble 反而最低（60.61%），D-S 融合在轴承二分类上起反作用

2. **变速轴承二分类（Ottawa 36样本）**：
   - 🏆 **Ensemble 达到 100% 完美**，D-S 融合在变速工况下发挥最大优势
   - MCKD 最佳单方法（83.33%），Teager 同分（83.33%，已修复耗时 Bug）
   - DWT 变速下仍保持 77.78%，是最鲁棒的轻量方法
   - VMD 从恒速 88.89% 降至变速 66.67%，对变速敏感

3. **齿轮诊断（WTgearbox 80样本）**：
   - 标准边频分析 78.75%、高级综合 80.00%、Ensemble 80.00%，三者基本持平

4. **多分类任务**：准确率普遍偏低——HUST 五分类 30.00%、WT 五分类 30.00%。所有非健康样本几乎都被预测为单一类别（外圈或健康），说明 Ensemble 的标签映射和 BPA 分配逻辑需要根本性重构。

5. **齿轮诊断（80样本）**：标准边频分析和高级综合均达到 78-80%，Ensemble 持平。但行星齿轮箱的部分故障类型（断齿、缺齿）在多分类中检出率不足。

6. **变速工况（Ottawa）**：Ensemble 集成方法优势最明显（~97% vs 单方法 ~83%）。DWT/EMD/CEEMDAN 等固定频带方法在变速工况下性能大幅下降。

7. **去噪效果**：VMD 单独去噪最好（+3.18dB），级联反而更差——级联实现需排查。MED 在所有 SNR 级别（含 -5dB）均检出故障，鲁棒性最佳。

8. **改进方向**（已在项目研究进展中定位根因）：
   - 引入直接多分类器替代二分类投票 → 解决五分类 30% 问题
   - 重构 D-S 证据融合 BPA 分配逻辑 → 解决标签偏向
   - 增加窄带包络与 VMD 频带细化 → 提升变速工况
   - 优化行星齿轮箱频域指标阈值 → 提升断齿/裂纹检出率
