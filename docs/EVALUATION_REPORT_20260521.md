## 4.7 去噪效果对比

对 HUSTbear 外圈故障 + 健康样本分别添加 0dB AWGN，测试 5 种去噪方法的 ΔSNR。

| 方法 | 外圈故障 ΔSNR(dB) | 健康 ΔSNR(dB) |
|------|-------------------|---------------|
| 无去噪 | 0.00 | 0.00 |
| 小波去噪 | +0.24 | +0.00 |
| VMD去噪 | **+3.18** | **+3.18** |
| Savitzky-Golay | −0.12 | +0.41 |
| 小波+VMD级联 | **+0.98** | +0.00 |

> ✅ 修复后级联从 +0.19dB 提升至 +0.98dB（提升 5×）。根因是小波软阈值过度去噪（`threshold_scale` 隐式为 1.0）扭曲了信号结构，导致 VMD 对"已失真信号"无法有效恢复。修复：级联默认 `threshold_scale=0.5` + VMD IMF 筛选以原始信号为参考。
>
> ⚠️ 级联仍不及 VMD 单独（+3.18dB），因为小波前置去噪会永久移除部分频带能量（含噪声+边缘有用信号），VMD 对"已失真"输入的恢复能力有限。

## 4.8 结果评述

从以上实验结果可以看出：

1. **恒速轴承二分类（HUSTbear 99样本）**：
   - 🏆 **VMD 85.86% 最佳单方法**，Ensemble 84.85% 紧随其后
   - 谱峭度重加权 72.73% 排名第三，频域自适应方法有优势
   - **Ensemble 从 60.61%→84.85%（Bug #15 修复）**，提升 24pp
   - DWT/标准包络/EMD 均在 64-69% 区间，差距缩小

2. **变速轴承二分类（Ottawa 36样本）**：
   - 🏆 **Ensemble 达到 100% 完美**，D-S 融合在变速工况下发挥最大优势
   - MCKD 最佳单方法（83.33%），Teager 同分（83.33%，已修复耗时 Bug）
   - 标准包络从恒速 64.65% 升至变速 69.44%，对变速不敏感

3. **齿轮诊断（WTgearbox 80样本）**：
   - 标准边频分析 78.75%、高级综合 80.00%、Ensemble 80.00%，三者基本持平

4. **多分类任务**：HUST 五分类 25.25%（Bal Acc 46.5%），WTG 五分类 **51.25%**（Bug #13 修复后 ×2.5 提升）。断齿检出仍是瓶颈（1/16，#13d）。

5. **去噪效果**：VMD 单独最好（+3.18dB），级联修复后 +0.98dB。MED 在所有 SNR 级别（含 -5dB）均检出故障，鲁棒性最佳。

6. **改进方向**（已在项目研究进展中定位根因）：
   - 引入直接多分类器替代二分类投票 → 解决五分类 30% 问题
   - 重构 D-S 证据融合 BPA 分配逻辑 → 解决标签偏向
   - 增加窄带包络与 VMD 频带细化 → 提升变速工况
   - 优化行星齿轮箱频域指标阈值 → 提升断齿/裂纹检出率
