## 4.7 去噪效果对比

对 HUSTbear 外圈故障 + 健康样本分别添加 0dB AWGN，测试 5 种去噪方法的 ΔSNR。

| 方法 | 外圈故障 ΔSNR(dB) | 健康 ΔSNR(dB) |
|------|-------------------|---------------|
| 无去噪 | 0.00 | 0.00 |
| 小波去噪 | +0.24 | ⏳ |
| VMD去噪 | +3.18 | ⏳ |
| MED去噪 | ⏳ | ⏳ |
| 小波+VMD级联 | +0.19 | ⏳ |

> ⚠️ 当前去噪 ΔSNR 偏低（最高 +3.18dB）。可能原因：
>
> 1. 使用单一样本（0.5X_O_20Hz-X.npy），信号本身噪声水平较低，噪声基底已接近量化噪声
> 2. ΔSNR 计算公式为 `10*log10(var(clean)/var(clean-denoised))`，对小幅改善不敏感
> 3. 级联去噪（小波+VMD）结果反而不如单独 VMD（+0.19 vs +3.18），需排查级联实现
> 4. 建议改用多文件平均 + 标准 SNR = `10*log10(var(signal)/var(noise))` 重新评估

## 4.8 结果评述

从以上实验结果可以看出：

1. **恒速轴承二分类（HUSTbear 99样本）**：
   - 标准包络、MED 增强、MCKD 均达到 **78-82%**，为最佳单方法组
   - Kurtogram（66.67%）和 CPW（66.67%）在 99 样本下表现不佳
   - **大样本（99）vs 小样本（20）结果差异显著**：MED 从 60%→78.79%，MCKD 从 50%→78.79%，说明小样本评估会严重低估实际性能

2. **多分类任务**：准确率普遍偏低——HUST 五分类 30.00%、WT 五分类 30.00%。所有非健康样本几乎都被预测为单一类别（外圈或健康），说明 Ensemble 的标签映射和 BPA 分配逻辑需要根本性重构。

3. **齿轮诊断（80样本）**：标准边频分析和高级综合均达到 78-80%，Ensemble 持平。但行星齿轮箱的部分故障类型（断齿、缺齿）在多分类中检出率不足。

4. **变速工况（Ottawa）**：Ensemble 集成方法优势最明显（~97% vs 单方法 ~83%）。DWT/EMD/CEEMDAN 等固定频带方法在变速工况下性能大幅下降。

5. **去噪效果**：当前 ΔSNR 偏低（最高 +3.18dB），需排查级联去噪实现 + 改用标准 SNR 公式重新评估。

6. **改进方向**（已在项目研究进展中定位根因）：
   - 引入直接多分类器替代二分类投票 → 解决五分类 30% 问题
   - 重构 D-S 证据融合 BPA 分配逻辑 → 解决标签偏向
   - 增加窄带包络与 VMD 频带细化 → 提升变速工况
   - 优化行星齿轮箱频域指标阈值 → 提升断齿/裂纹检出率
